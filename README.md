# EnglishMaster

EnglishMaster is an interactive platform designed to help users improve their English speaking, grammar, vocabulary, and confidence through structured lessons and daily practice.

## 🚀 Features
- Grammar lessons
- Vocabulary building
- Interactive exercises
- Clean and responsive design

## 🛠 Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS

## 📦 Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## 🌐 Future Improvements
- User authentication
- Progress tracking
- Quiz system
